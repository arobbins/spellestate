== Description ==

Export subscriptions details out of WooCommerce into a CSV-formatted file.

== Installation ==

1. Upload the folder 'wc-subs-exporter-ppgr' to the '/wp-content/plugins/' directory
2. Activate 'Subscription Exporter' through the 'Plugins' menu in WordPress

== Usage ==

1. Open WooCommerce > Subscriptions Export from the WordPress Administration
1. Select which WooCommerce details you would like to export and click Export

== Screenshots ==

1. Select the data you wish to export.
